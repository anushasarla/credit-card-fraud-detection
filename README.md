# credit-card-fraud-detection
A machine learning project that detects fraudulent credit card transactions by analyzing imbalanced financial data and building a classification model for accurate fraud detection.


## Dataset
The dataset used in this project is available on Kaggle:

🔗 https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Download the dataset manually and place it in the `data/` folder before running the notebook.
